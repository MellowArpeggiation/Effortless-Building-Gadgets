# Effortless Builing Gadgets
> Build impressive structures, all in survival!

Hold left alt while holding the gadget to configure.

It is highly recommended that you play with [Baubles Expanded](https://github.com/GTNewHorizons/Baubles-Expanded) installed. With it present it can be placed in a charm slot, which will allow it to be used and configured while holding any placeable block in your hotbar!

## Capabilities
Place blocks in all sorts of different shapes, automatically replacing any soft blocks like grass or snow, shapes available include:
* Extended (regular placing, with greatly increased reach)
* Air (Place blocks directly in the air, without the need to make a pillar up first)
* Lines
* Floors + Walls
    * Both Filled and Hollow
* Cubes
    * Filled, Hollow, and Skeletonised options available

Will not function with any blocks that place with tile entities. (Some exceptions coming soon, as noted in planned features)

Requires power from:
* **RF** (CoFH)
* **HE** (NTM)

## Planned features
* Cylinder, circle, and sphere shapes
* Diagonal lines and walls
* Support for ArchitectureCraft blocks
* Cut/Copy/Paste gadgets (+rotation!)
* (When in Bauble slot) use tools like pickaxes/shovels for shape breaking
* Shape breaking blocks placed in session (or any, if in creative)
* Modifiers from Effortless like Mirror, Array, etc
* AE2 compatibility (pulling blocks from storage)
    * Potentially compatibility for any mods adding handheld storage + backpacks?
* Chisel compatibility (automatically chiseling blocks in inventory/storage to place)
* More configuration for existing tools (air placement distance, etc)
* Block randomisation
* Support for more power sources
